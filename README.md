# Habit Tracker

A simple, beautiful habit tracker that runs in your browser. Data is saved locally — no account required.

## Features

- Add habits with emoji, color, and optional notes
- Quick-add presets (workout, read, meditate, etc.)
- One-tap check-off with confetti when all habits are done
- Edit, reorder, search, and sort habits
- 7-day dots + expandable 30-day heatmap per habit
- Monthly calendar overview
- Weekly stats (check-ins, best streak, perfect days)
- Light / dark theme
- Export & import JSON backups
- Undo delete
- Keyboard shortcut: `N` to add a habit

Note : This was built using Cursor AI to experience what vibe coding is
